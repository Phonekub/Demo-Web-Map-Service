Follow this rules when coding:

- Always use best practice of React and TS
- Consider performance (not re-render with unnecessary)
- Not use deprecated function
- Make clean code, easy to read and maintain
- Create UXUI easy to use and understand
- CSS Component from tailwind and daisyUI
